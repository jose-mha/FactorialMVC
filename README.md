### Descripción

- Calcula el factorial de un número.
- Usando MVC ASP.NET Core

### Ejemplo
----

### Factorial de 4

### Factorial de 4  = 24


| -  | -  |  - | - | - |
| ------------- | ------------- | ------------- | ------------- | ------------- |
| 1 | * | 1 | = | 1 |
| 1 | * | 2 | = | 2 |
| 2 | * | 3 | = | 6 |
| 6 | * | 4 | = | 24 |
----
